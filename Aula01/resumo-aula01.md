
## O que é um algoritmo (I.A.)
Algoritmo é uma sequência de passos organizados para resolver um problema
ou realizar uma tarefa do começo ao fim.

## O que é um algoritmo
Git é uma ferramenta que serve para salvar diferentes verções de seu codigo usada para registrar as alterações feitas em arquivos ao longo do tempo.
Com ele, é possível acompanhar o histórico do trabalho e evitar perda de conteúdo.
É uma plataforma online onde os projetos são armazenados e compartilhados.
Em aula, vamos usar Git como ferramenta de organização, registrar nosso aprendizado
e já nos preparar para o ambiente profissional.
